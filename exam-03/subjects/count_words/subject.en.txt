Assignment name  : count_words
Expected files   : count_words.c
Allowed functions: 
Version          : 1
--------------------------------------------------------------------------------

Write a function that takes a string, splits in into words, and returns an integer
for the number of words.

A "word" is defined as part of a string delimited by spaces, or by the start/end
of the string.

Your function must be declared as follows:

int     count_words(char *str);
